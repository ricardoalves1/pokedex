## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Environment Variables

You must create a file called `.env.local` or rename the file "`.env.example`". It must contain the following key:

```
API_BASE_URL
```
